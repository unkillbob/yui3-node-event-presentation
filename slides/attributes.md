##  Attributes

```javascript
// Set valid HTML attribute
myNode.set('title', 'Hover to see me!');
myNode.set('data-foo', 'bar');
```

```javascript
// Set custom attributes
myNode.setAttribute('custom-attr', '!');
```

```javascript
// Set multiple attributes
myNode.setAttrs({
    title: 'A title',
    'data-foo': 'bar'
});
```
